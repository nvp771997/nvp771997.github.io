Hàm fseek và ftell

Hàm fseek

Hàm fseek được sữ dụng để di chuyển con trỏ file đến vị trí mong muốn



        Syntax
        int fseek(FILE *pointer, long int offset, int position);
  • pointer: con trỏ file
  • offset: số lượng byte di chuyển
  • position: vị trí bắt đầu di chuyển ->
    • SEEK_END : Bắt đầu từ cuối file
    • SEEK_SET : Bắt đầu từ đầu file
    • SEEK_CUR : Bắt đầu từ vị trí hiện tại

Hàm ftell

Hàm ftell được sử dụng để chỉ ra vị trí của con trỏ file hiện tại, được tính bằng số byte từ đầu file từ vị trí hiện tại của con trỏ file.



        Syntax
        ftell(FILE *pointer);
     


        
       #include < stdio.h>
       #include < string.h>
       int main(){
          FILE *f;
          f = fopen("input.txt", "r"); //Read
           if(f == NULL){
              printf("Khong ton tai\n");
           }
           else{
             fseek(f, 0, SEEK_END); //Con trỏ bắt đầu ở cuối file
             fseek(f, 0, SEEK_SET); //Con trỏ bắt đầu ở đầu file
             fseek(f, 4, SEEK_SET); //Con trỏ bắt đầu ở đầu file và ở vị trí 4 byte
             fseek(f, 3, SEEK_CUR); //Con trỏ bắt đầu ở vị trí hiện tại
             printf("%d\n", ftell(f)); //Từ vị trí đầu file đến vị trí của con trỏ hiện tại là bao nhiêu byte
           }
          fclose(f);
       }

Những cái cơ bản

Data type

Data type (kiểu dữ liệu) được dùng để định dạng cho biến

Các kiểu dữ liệu:

Kiểu số nguyên
Data type Meaning Size Format
short Số nguyên có dấu 2 byte, -32768 đến 32767 %d, %hd
unsigned short Số nguyên không dấu 0 đến 65535 %d, %hd
int Số nguyên có dấu 4 byte, -2147483648đến 2147483647 = -2*10^9 đến -2*10^9 %d
unsigned int Số nguyên không dấu 4 byte, 0 đến 4294967295 = 4*10^9 %u
long long Số nguyên có dấu 8 byte, -9223372036854775808 đến 9223372036854775807 = -9*10^18 đến 9*10^18 %lld
unsigned long long Số nguyên không dấu 8 byte, 0 đến 2^64 -1 %llu


Kiểu số thực
Data type Meaning Size Format
float Độ chính xác đơn 4 byte, 1.17549e-038 đến 3.40282e+038Độ chính xác : 7 chữ số sau dấu phẩy %lf
double Độ chính xác kép 8 byte, 2.22507e-308 đến 1.79769e+308Độ chính xác : 15 chữ số sau dấu phẩy %llf

Variable

Variable (Biến) được dùng để lưu trữ các giá trị để chương trình sữ dụng

Tất cả các biến phải có tên và phải có kiểu dữ liệu dứng trước thì chương trình mới thực thi được
